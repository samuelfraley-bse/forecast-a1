## Slide 2
Choose one application

You will work with one of three text corpora.

1. Financial PhraseBank (sentiment)

• Short financial sentences labeled as negative / neutral / positive

• Suggested: text classification, benchmarking, few-shot fine-tuning

2. ACLED notes (events)

• Use text to enrich event characterization (Spain, Sudan)

• Suggested: build an index (repression / escalation), classify violent vs non-violent

3. Panel forecasting (macro + text)

• Predict recession risks using structured + unstructured data

• Suggested: build text-based signals, evaluate forecasting performance
